# 全链路数据资产监控平台 🔍

> 5层穿透式数据资产监控 + 字段级血缘关系可视化

## 产品定位

本平台旨在解决数据治理中的"数据看不清、血缘找不到、影响不知道"三大痛点，提供从**服务器 → 应用 → 数据库 → 表 → 字段 → 表间关系**的6层穿透式监控能力。

## 核心功能

- 🔍 **5层穿透导航**: 服务器 → 应用 → 数据库 → 表 → 字段
- 🔗 **字段级血缘可视化**: 基于 AntV G6 的交互式血缘图
- ⚡ **影响分析**: 点击字段，一键查看上下游影响范围
- 📊 **实时状态监控**: CPU/内存仪表 + 健康度统计
- 🔎 **全局搜索**: 表名/字段名秒级检索

## 技术架构

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| 前端框架 | Vue 3 | Composition API |
| 状态管理 | Pinia | 轻量级 Vue 3 专用 |
| UI组件 | Element Plus | 国产最优 |
| 图可视化 | AntV G6 | 阿里开源，强大易用 |
| 数据模拟 | JSON | mock-data.json |

## 快速运行

```bash
# 1. 进入项目目录
cd data-asset-monitor-fullstack

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev

# 4. 浏览器打开
# http://localhost:5173
```
